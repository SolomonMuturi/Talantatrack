# Talantatrack

A comprehensive sports talent management and team administration mobile application built with Flutter.

## Features

- **Player Management** - Enroll, view, and manage player profiles
- **Event Management** - Create and schedule team events and matches
- **Financial Management** - Track revenue, expenses, and payments
- **Dashboard Analytics** - View KPIs, charts, and recent transactions
- **Cross-Platform** - Runs on Android, iOS, Web, Windows, macOS, and Linux
